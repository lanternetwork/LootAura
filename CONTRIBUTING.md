# Contributing

LootAura is **proprietary software** owned by Lanternetwork.

## External contributions

External contributions are **not accepted**. This includes pull requests, patches, and unsolicited code or documentation changes from outside the organization.

## Security reports

If you discover a security vulnerability, follow the process in [SECURITY.md](SECURITY.md). Do not open public issues for security concerns.

## Contact

- **Security:** security@lootaura.com (see [SECURITY.md](SECURITY.md))
- **Licensing:** contact Lanternetwork

## Team development

Authorized Lanternetwork developers should follow internal engineering standards and the documentation in [docs/README.md](docs/README.md).
