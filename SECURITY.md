# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, **do not** open a public GitHub issue.

Email **security@lootaura.com** with:

- A description of the issue
- Steps to reproduce (if applicable)
- Potential impact
- Suggested remediation (optional)

## What to expect

- We aim to acknowledge reports within **48 hours**
- We will work with you to understand and address valid findings
- Please allow reasonable time for investigation and remediation before public disclosure

## Scope

Reports related to the LootAura web application, mobile apps, and publicly deployed services are in scope.

## Out of scope

- Social engineering
- Denial-of-service attacks
- Issues in third-party services outside our control
- Reports requiring physical access to user devices

Thank you for helping keep LootAura secure.
